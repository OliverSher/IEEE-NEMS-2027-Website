# IEEE NEMS 2027 — 開發待辦 & 資料準備清單

> 與 IEEE NEMS 2024 網站比較後整理出的差距分析與待辦事項。
> ✅ = 已完成 | 🔧 = 可由開發端處理 | 📋 = 需要主辦方提供資料

## 一、需要主辦方準備的資料

以下是網站開發**需要你提供**的內容，建議按優先順序準備：

### 🔴 高優先（網站上線必備）

| # | 項目 | 說明 | 格式建議 |
|---|------|------|---------|
| 1 | **會議日期** | 確切的會議日期（幾月幾日到幾日） | 文字 |
| 2 | **重要日期** | 摘要截止日、通知日、全文截止日、早鳥註冊截止日 | 文字 |
| 3 | **大會主席照片與資訊** | General Chair、Co-Chair 的肖像照、姓名、職稱、單位 | 照片 JPG/PNG（至少 300x400px） |
| 4 | **歡迎詞定稿** | 大會主席的正式歡迎信（目前有草稿，需確認或替換） | 中英文文字 |
| 5 | **委員會名單** | 所有委員會成員的姓名、職稱、機構、國家 | Excel 或文字 |
|   | — International Advisory Committee | 主席 + 約 5-8 位委員 | |
|   | — Steering Committee | 主席 + 約 10 位委員 | |
|   | — Organizing Committee | 各職位（約 10-15 個角色）| |
|   | — Technical Program Committee | 約 30-90 位委員 | |
| 6 | **聯絡資訊** | 秘書處 email、電話、聯絡人 | 文字 |
| 7 | **註冊費用** | 各類別（IEEE 會員/非會員/學生等）的早鳥、標準、現場費率 | 文字或表格 |

### 🟡 中優先（徵稿期間需要）

| # | 項目 | 說明 | 格式建議 |
|---|------|------|---------|
| 8 | **投稿系統連結** | SoftConf 或其他投稿平台的 URL | URL |
| 9 | **註冊系統連結** | 線上註冊付款平台的 URL | URL |
| 10 | **Call for Papers 海報** | 可分享的 CFP 圖文海報（用於社群和 email 散佈） | PDF + PNG（A4 或方形） |
| 11 | **Banner 主視覺圖片** | 網站橫幅用，建議含清華大學/新竹意象 | 3000×1500px 或 2:1 比例 |
| 12 | **組織單位 Logo** | IEEE、IEEE NTC、清華大學的官方 Logo | PNG/SVG（透明背景） |
| 13 | **簽證資訊** | 台灣簽證申請說明、外交部連結、簽證協助聯絡窗口 | 文字 + URL |
| 14 | **付款方式** | 接受的付款方式（信用卡、匯款等） | 文字 |

### 🟢 後期準備（會議前 2-3 個月）

| # | 項目 | 說明 | 格式建議 |
|---|------|------|---------|
| 15 | **Plenary Speakers** | 每位講者：照片、姓名、職稱、機構、講題、摘要、簡歷、個人網頁 | 照片 + 文字 |
| 16 | **Keynote Speakers** | 同上 | 照片 + 文字 |
| 17 | **Invited Sessions** | 每場次：場次代碼、標題、主持人、說明、各演講者與講題 | 文字或 Excel |
| 18 | **議程表** | Program at a Glance（時間表格）| PDF + 圖片 |
| 19 | **Technical Tour 資訊** | 參訪地點、行程、費用、人數限制、報名連結 | 文字 + 照片 |
| 20 | **贊助商資料** | 各級贊助商名稱、Logo、網址 | Logo PNG + URL |
| 21 | **場地照片** | 清華大學校園、旺宏館、會議廳、成功湖等 | JPG（至少 1200px 寬） |
| 22 | **新竹觀光照片** | 城隍廟、十八尖山、海岸線等特色景點 | JPG |
| 23 | **住宿合作飯店** | 如有議價合作的飯店，提供名稱、價格、訂房連結 | 文字 + URL |
| 24 | **社群媒體帳號** | Twitter/X、Facebook 等（如有建立） | URL |

## 四、建議準備順序

```
第一階段（立即）
  → 確定會議日期、重要日期
  → 確定聯絡資訊
  → 提供委員會名單
  → 提供註冊費用

第二階段（1-2 個月內）
  → 提供 Banner 主視覺 / 或授權設計
  → 提供大會主席照片 + 正式歡迎詞
  → 建立投稿系統 + 註冊系統
  → 整理簽證資訊
  → 準備組織單位 Logo

第三階段（會議前 3 個月）
  → 確定講者名單及資料
  → 確定 Invited Sessions
  → 確定贊助商
  → 規劃 Technical Tour

第四階段（會議前 1 個月）
  → 議程表定稿
  → 場地 / 觀光照片
  → 最終更新
```
